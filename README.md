# Imersão-Dados-Python-Alura-2026

Projeto desenvolvido durante a Imersão Dados com Python, oferecida pela Alura.

O projeto utiliza Python para a construção de um dashboard interativo, empregando as bibliotecas Streamlit, Pandas, Matplotlib e Plotly.
Foram criados gráficos estáticos, estatísticos e interativos, com foco na exploração e visualização de dados.
